# Prólogo

Este libro es para desarrolladores que ya conocen TypeScript y React, y quieren integrar inteligencia artificial en sus aplicaciones **sin cambiar de stack**.

## ¿Por qué TypeScript para IA?

La mayoría de tutoriales y cursos de IA asumen que necesitas Python. Notebooks de Jupyter, bibliotecas como LangChain en Python, frameworks como FastAPI. Pero si eres desarrollador web, esto significa aprender un lenguaje nuevo, configurar entornos virtuales, y mantener dos stacks separados.

**El AI SDK de Vercel cambia esto.**

Es una biblioteca TypeScript-first que te permite:
- Hacer streaming de respuestas de LLMs en tiempo real
- Generar datos estructurados con validación de tipos
- Crear agentes con herramientas personalizadas
- Todo con la misma sintaxis y patrones que ya usas en React

## ¿Qué aprenderás?

Este libro te llevará desde los fundamentos hasta técnicas avanzadas:

1. **Tu Primera Inferencia**: Cómo funcionan los LLMs, tokens, y contexto
2. **React y useChat**: El hook que hace trivial construir interfaces de chat
3. **Streaming**: Cómo funciona el protocolo de streaming por dentro
4. **React Router v7**: Tu chat full-stack con el framework moderno
5. **Structured Output**: Obtener datos tipados con validación Zod
6. **Tools**: Cuando el modelo ejecuta acciones en tu sistema
7. **Agentes**: Sistemas autónomos que razonan y actúan
8. **Generación de Imágenes**: Crear visuales con código
9. **Embeddings**: Búsqueda semántica y vectores
10. **RAG**: Retrieval Augmented Generation para chatbots con conocimiento
11. **Agentic RAG**: Agentes que deciden qué buscar
12. **Audio y Speech**: Voz e IA

## Para quién es este libro

- Desarrolladores React/TypeScript que quieren añadir IA
- Desarrolladores Python curiosos sobre el ecosistema JavaScript
- Cualquiera que quiera construir aplicaciones de IA sin salir de la web

No necesitas experiencia previa con IA o machine learning. Solo necesitas saber TypeScript y tener curiosidad.
